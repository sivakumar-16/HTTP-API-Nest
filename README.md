# festivalData

## project overview

This task is to creare a HTTP API application that lists out music festival data in a particular manner.

## technology used

- node
- Typescipt
- NestJs
- Docker
- jest
 
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
